
#Commands For Git :
--------------------
--------------------


## git clone : هذا الامر يحضر اي repo من اي server مثل ( githup )
----------

## git add : نضع بعد هذا الامر اسم الملف .. هذا الامر نستخدمه بعد التعديل على الملف الذي جلبناه من githup مثلا لكي نضعه في stage area  وهي منطقة ترفع فيها الملفات قبل ان نضعها في local area 
----------

## git reset head : هذا الامر لكي نرجع الملف الذي وضعناه في stage area ولا نريد ان ننقله الى مرحلة local area بسبب خطأ فيه مثلا ..
----------------

## git commit -m : هذا الامر نضع بعده علامتين تنصيص " " و نكتب بينهما معلومات عن الملفات التي نرفعها حيث ان هذا الامر نستخدمه لكي ننقل الملفات من ال stage area الى local area 
----------------

## git branch : هذا الامر نستخدمه لكي نعرف اسم الفرع الذي نستخدمه للتعديل على الملفات التي استوردناها من githup مثلا او اي server نستخدمه
----------------

## git remote -v : هذا الامر نستخدمه لكي نعرف اسم ال repo الذي سنرفع عليه الملفات من ال local الى ال server
----------------

## git push : هذا المر نستخدمه لارسال الملفات من ال local  الى  repo حيث نضع بعد هذا الامر اسم ال repo و بعده اسم ال branch
----------------
